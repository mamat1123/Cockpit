# Claude Cockpit — M11: Theming (10 themes) + accent + blur

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use `- [ ]`.

**Goal:** 10 switchable color themes + an accent override + a blur control, all live + persisted, in the Settings menu (which already has the opacity slider). Default = Amber HUD (the current look — the app must look identical until another theme is chosen).

**Architecture:** All app colors become CSS custom properties (`--ck-*`) on `:root` (defaults = Amber HUD). A theme = a token map; `applyTheme()` writes the tokens onto `document.documentElement`. xterm + ECharts read the active theme's colors. Settings persists `{ themeId, accent, blur, bgOpacity }`. Blur = a macOS window-effect material applied live via a Rust `set_window_effect` command.

**Tech Stack:** React 19, CSS custom properties + `color-mix()`, Rust (window.set_effects), vitest.

## Token set (defined once, on `:root`)
`--ck-bg` (darkest: terminal/pane/active-tab/overlay panels), `--ck-surface` (chrome: tab bar/headers), `--ck-surface-2` (hover/badge/input), `--ck-text`, `--ck-bright` (headings), `--ck-muted`, `--ck-dim` (labels/tertiary), `--ck-border` (hairlines), `--ck-accent` (brand/working), `--ck-idle` (success/idle), and ANSI-ish `--ck-blue --ck-green --ck-red --ck-yellow --ck-magenta --ck-cyan`.

```ts
export const DEFAULT_THEME_ID = "amber-hud";
export function themeById(id: string): Theme { return THEMES.find((t) => t.id === id) ?? THEMES[0]; }

/** Write a theme's tokens (with optional accent override) onto the document root. */
export function applyTheme(theme: Theme, accent?: string | null): void {
  const r = document.documentElement.style;
  const set: Record<string, string> = {
    "--ck-bg": theme.bg, "--ck-surface": theme.surface, "--ck-surface-2": theme.surface2,
    "--ck-text": theme.text, "--ck-bright": theme.bright, "--ck-muted": theme.muted, "--ck-dim": theme.dim,
    "--ck-border": theme.border, "--ck-accent": accent || theme.accent, "--ck-idle": theme.idle,
    "--ck-blue": theme.blue, "--ck-green": theme.green, "--ck-red": theme.red,
    "--ck-yellow": theme.yellow, "--ck-magenta": theme.magenta, "--ck-cyan": theme.cyan,
  };
  for (const [k, v] of Object.entries(set)) r.setProperty(k, v);
}
```

```rust
#[tauri::command]
fn set_window_effect(window: tauri::WebviewWindow, material: String) {
    use tauri::window::{Effect, EffectsBuilder};
    let eff = match material.as_str() {
        "hudWindow" => Some(Effect::HudWindow), "fullScreenUI" => Some(Effect::FullScreenUI),
        "sidebar" => Some(Effect::Sidebar), "underWindowBackground" => Some(Effect::UnderWindowBackground),
        _ => None,
    };
    let _ = match eff {
        Some(e) => window.set_effects(EffectsBuilder::new().effect(e).build()),
        None => window.set_effects(EffectsBuilder::new().build()), // "none" → clear
    };
}
```
