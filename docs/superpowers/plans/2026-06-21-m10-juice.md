# Claude Cockpit — M10: Juice (workflow-rhythm, not keypress-combo)

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use `- [ ]`.

**Goal (ADR 0002 — pure visual juice, no scoring/mechanics):** make the cockpit *feel* alive, triggered by the REAL Claude rhythm (one prompt → long async response), not by input frequency:
- **Launch flash** when you send a prompt (Enter).
- **Swarm meter** — a HUD whose intensity escalates with the number of panes currently *working* concurrently (this is the "combo" the user liked, retargeted to real parallelism).
- **Completion pull-back** — when a pane finishes (working→idle) and it's in a tab you're NOT looking at, that tab glows to pull you back; clears when you view it.
- Thinking ambient stays the existing breathing vignette (M2).

**Architecture:** a `juiceBus` (send events from the registry's keystroke handler) + a single `<Juice>` overlay that polls per-pane state from the terminal registry (`paneLastLineAt` + `deriveState`), counts working panes (swarm), and detects working→idle transitions (completion → tab attention via a callback). Tab attention lives in CockpitView; TabBar renders the glow; viewing a tab clears it.

**Tech Stack:** React 19 · vitest. (Tool-use blips need log-event parsing → deferred.)

- [ ] **Step 2: implement** `src/lib/juiceBus.ts`:
```ts
type Cb = () => void;
const subs = new Set<Cb>();
/** Subscribe to "prompt sent" events; returns an unsubscribe fn. */
export function onSend(cb: Cb): () => void { subs.add(cb); return () => { subs.delete(cb); }; }
/** Fire a "prompt sent" event (called when the user presses Enter in a pane). */
export function emitSend(): void { subs.forEach((cb) => cb()); }
```

## Task 2: `<Juice>` overlay — launch flash + swarm meter + completion detect

**Files:** create `src/components/Juice.tsx`, `src/components/Juice.css`.

- [ ] **Step 1: `Juice.tsx`:**
```tsx
import { useEffect, useRef, useState } from "react";
import type { Layout } from "../layout/paneLayout";
import { paneLastLineAt } from "../lib/terminalRegistry";
import { deriveState } from "../lib/paneState";
import { onSend } from "../lib/juiceBus";
import "./Juice.css";

export function Juice({ layout, onAttention }: { layout: Layout; onAttention: (tabId: string) => void }) {
  const [working, setWorking] = useState(0);
  const [flash, setFlash] = useState(0);
  const layoutRef = useRef(layout); layoutRef.current = layout;
  const onAttentionRef = useRef(onAttention); onAttentionRef.current = onAttention;
  const prev = useRef<Map<string, boolean>>(new Map());

  useEffect(() => onSend(() => setFlash((f) => f + 1)), []);

  useEffect(() => {
    const id = setInterval(() => {
      const l = layoutRef.current;
      const now = Date.now();
      let count = 0;
      const live = new Set<string>();
      for (const t of l.tabs) for (const r of t.rows) for (const p of r.panes) {
        live.add(p.id);
        const w = deriveState({ lastLineAt: paneLastLineAt(p.id) }, now, 800) === "working";
        if (w) count++;
        const was = prev.current.get(p.id) ?? false;
        if (was && !w && t.id !== l.activeTabId) onAttentionRef.current(t.id); // completion in a background tab
        prev.current.set(p.id, w);
      }
      for (const id of [...prev.current.keys()]) if (!live.has(id)) prev.current.delete(id);
      setWorking(count);
    }, 350);
    return () => clearInterval(id);
  }, []);

  const lv = Math.min(working, 6);
  return (
    <div className="juice" aria-hidden="true">
      {flash > 0 && <div key={flash} className="juice__launch" />}
      {working > 0 && (
        <div className={`juice__swarm lv${lv}`}>
          <span className="juice__bolt">⚡</span>
          <b>{working}</b>
          <span className="juice__lbl">working</span>
        </div>
      )}
    </div>
  );
}
```
